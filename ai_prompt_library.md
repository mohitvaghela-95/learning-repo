AI prompt library

This file is used to collect useful prompts for reuse when working with AI.

A simple formula, you should remember to use when writing prompts is:
  Role + Task + Context + Constraints + Output (RTCCO) 
  Real Tigers Cant Catch Octopi - is a way you can remember.

For rewriting
1) Spellcheck

You are a spell-checker tool. Your job is to check my the following input text and look for spelling mistakes and gramatical errors. Then you should correct them.

[INPUT TEXT HERE]

If you identify text that doesn't make sense or is hard to understand or can be written more concisely, do not alter it but point it out and make ONE suggestion per instances of text flagged.

Your output should be as follows:
- Any suggestions (if applicable)
- Original text with the spelling and grammar mistakes highlighted
- Corrected text
- How I would like to proceed if there are any suggestions.
